# AgriPulse Precision Agriculture

AgriPulse is an AI-powered web application that analyzes soil nutrient data and provides fertilizer recommendations.

## Features
- Upload soil CSV file
- Crop suitability scoring
- Nutrient deficiency detection
- Fertilizer recommendations
- AI-generated explanations
- Farmer-friendly color dashboard

## Supported Crops
- Tomato
- Wheat
- Rice
- Maize

## Setup

Install dependencies

pip install -r requirements.txt

Run application

python app.py

Open browser

http://127.0.0.1:5000

## CSV Format

soil_id,nitrogen,phosphorus,potassium,ph_level
